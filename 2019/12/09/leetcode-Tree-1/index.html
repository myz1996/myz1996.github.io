#257 Binary Tree Paths

本题思路，从根结点一直往下走到底。那么首先应该考虑深度优先。需要注意的是，什么时候把String添加到list中呢，当访问到没有左子结点也没有右子结点的结点时。开始在if判断中，我写的是

if (node.left 1= null && node,right != null)

这样就会错过很多路径，应该是只要左右子结点有一个不为空就可以继续向下挖掘。
