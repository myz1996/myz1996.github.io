#75 Sort Colors

第一种方法：循环两次。第一次循环统计三个数分别出现的次数，第二次循环直接按照相应的区间重新赋值数组。不用多此一举考虑原来的数值是否正确。

第二种方法：三路快排。采用的是三个指针的方式，一个指针i用于遍历。left，right分别从左右向中间逼近。

为0的放左边，与left指针指向的位置交换，2放右边与right指针交换。需要注意的是，与左指针交换后，left和i都加一，但是与右指针交换后i不变，因为交换回来的值仍需要一次判断。

深入理解三路快排！快排真的很快！

#79 Word Search

本题先来分析一下，首先应该遍历二维数组找到一个元素与String第一个元素相同，从这个元素开始向上下左右分别找是否接下来的元素一一对应。需要注意的是，我们不能使用同一个元素两次，怎样实现呢？对于数组来说我们可以使用一个辅助的boolean二维数组来记录某个元素是否被访问过。

递归的结束：当String最后一个元素也被比较完毕的时候。

i， j是否越界，是则返回false。如果上下左右已经被访问过则也要返回false，比较的元素不相同返回false。将当前元素设置为访问过，然后上下左右分别递归调用check方法来检查，只要有一个路径满足，整体可以返回true。否则要返回false，说明以某个点开始的上下左右路径没有符合要求的。应当在二维数组中继续遍历找到一个头元素相同的位置。别忘记访问过的标记在这种情况下应当重新置为false，因为没有找到。
