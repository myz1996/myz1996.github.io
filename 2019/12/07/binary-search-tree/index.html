二叉查找树

size(root) = size(left) + size(right) +1

最好的情况下，一棵含有N个结点的树是完全平衡的。

平衡二叉查找树：

  1. 可以是空树。
  2. 假如不是空树，任何一个结点的左子树与右子树都是平衡二叉树，并且高度之差的绝对值不超过 1。

在由N个随机键构造的二叉查找树中，插入操作，查找未命中、查找命中平均所需的比较次数为～2lnN

删除

后继节点

1.如果节点有右子树，则该节点的后继节点就是往右子树出发，然后转到右子树的左子树，一直到左子树的左子树为空

2.如果节点没有右子树，则向上寻找父节点，直到父节点的左子树等于当前节点，则该父节点就是后继节点

这个删除有点复杂，有点没明白。
